######################################################################## Backend-Installation ##############################################################
Requirements to be Installed:
	Windows 
	Python
	Postman 


1. Install Python Environment using "py -m venv env" in the directory where you placed survey file.
2. Install dependencies in the environment by activating the environment "./env/Scripts/activate.bat" and installing
following dependencies : "pip install Flask", "pip install Flask-Restful", "pip install Flask-JWT".
3. Run the app.py file from survey folder.
4. Run Postman and Import "CPSC671.postman_collection", and run each API endpoint.

######################################################################## Front-End-Installation ##############################################################

Front End Configuration.
1. To run the front end project you need to download
the Unity version 2020 or latest
2. Here is the link to download : https://unity3d.com/get-unity/download
3. Install the Unity on your PC
4. Open Unity
5. Click on the Add Button in UnityHub and provide the location of project.
6. After adding the Project to UnityHub Click on the project to open it.
7. Project will be opened now
8. To Run the Project Please follow all the step to Configuration of backend.
9. Run the server app locally in you PC where Unity is installed.
10. After the successfully running server click on the Play Button in Middle Top Center Unity Project.
11. You can test the project and it should run
